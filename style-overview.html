CSS Animationen Übersicht

Eine Sammlung der wichtigsten CSS-Animationen mit Code-Beispielen

Diese Seite zeigt verschiedene CSS-Animationen, die Sie in Ihren Webprojekten verwenden können. Klicken Sie auf die Buttons, um die Animationen zu starten oder zu stoppen.

Bounce

Ein springender Effekt, bei dem das Element nach oben schnellt und zurückprallt.

@keyframes bounce { 0%, 20%, 50%, 80%, 100% {transform: translateY(0);} 40% {transform: translateY(-30px);} 60% {transform: translateY(-15px);} }

Fade In

Das Element erscheint allmählich durch Erhöhen der Deckkraft.

@keyframes fadeIn { from {opacity: 0;} to {opacity: 1;} }

Slide In

Das Element gleitet von der Seite ins Blickfeld.

@keyframes slideIn { from {transform: translateX(-100px); opacity: 0;} to {transform: translateX(0); opacity: 1;} }

Rotate

Dreht das Element kontinuierlich um 360 Grad.

@keyframes rotate { from {transform: rotate(0deg);} to {transform: rotate(360deg);} }

Pulse

Erzeugt einen pulsierenden Effekt durch Skalierung des Elements.

@keyframes pulse { 0% {transform: scale(1);} 50% {transform: scale(1.1);} 100% {transform: scale(1);} }

Shake

Das Element bewegt sich schnell hin und her, wie bei einem Erschütterungseffekt.

@keyframes shake { 0%, 100% {transform: translateX(0);} 10%, 30%, 50%, 70%, 90% {transform: translateX(-10px);} 20%, 40%, 60%, 80% {transform: translateX(10px);} }

Flip

Das Element dreht sich um die Y-Achse, ähnlich einer Karte, die umgedreht wird.

@keyframes flip { 0% {transform: perspective(400px) rotateY(0);} 50% {transform: perspective(400px) rotateY(180deg);} 100% {transform: perspective(400px) rotateY(360deg);} }

Wobble

Ein wackelnder Effekt, bei dem das Element hin und her schwankt.

@keyframes wobble { 0% {transform: translateX(0%);} 15% {transform: translateX(-25%) rotate(-5deg);} 30% {transform: translateX(20%) rotate(3deg);} 45% {transform: translateX(-15%) rotate(-3deg);} 60% {transform: translateX(10%) rotate(2deg);} 75% {transform: translateX(-5%) rotate(-1deg);} 100% {transform: translateX(0%);} }

Flash

Das Element blinkt auf und ab durch schnelle Änderung der Deckkraft.

@keyframes flash { 0%, 50%, 100% {opacity: 1;} 25%, 75% {opacity: 0;} }

Zoom In

Das Element vergrößert sich von unsichtbar zu voller Größe.

@keyframes zoomIn { from {transform: scale(0); opacity: 0;} to {transform: scale(1); opacity: 1;} }

Gradient Shift

Ein sanfter Übergang zwischen Farbverlaufsfarben.

@keyframes gradientShift { 0% {background-position: 0% 50%;} 50% {background-position: 100% 50%;} 100% {background-position: 0% 50%;} }

Swing

Das Element schwingt wie ein Pendel hin und her.

@keyframes swing { 20% {transform: rotate(15deg);} 40% {transform: rotate(-10deg);} 60% {transform: rotate(5deg);} 80% {transform: rotate(-5deg);} 100% {transform: rotate(0deg);} }

Rubber Band

Das Element dehnt sich wie ein Gummiband und schnellt wieder zurück.

@keyframes rubberBand { 0% {transform: scaleX(1);} 30% {transform: scaleX(1.25) scaleY(0.75);} 40% {transform: scaleX(0.75) scaleY(1.25);} 50% {transform: scaleX(1.15) scaleY(0.85);} 65% {transform: scaleX(0.95) scaleY(1.05);} 75% {transform: scaleX(1.05) scaleY(0.95);} 100% {transform: scaleX(1);} }

Heartbeat

Ein rhythmischer Herzschlag-Effekt durch Skalierung.

@keyframes heartbeat { 0% {transform: scale(1);} 14% {transform: scale(1.3);} 28% {transform: scale(1);} 42% {transform: scale(1.3);} 70% {transform: scale(1);} }

Jello

Das Element schwingt wie Wackelpudding.

@keyframes jello { 0% {transform: none;} 11.1% {transform: skewX(-12.5deg) skewY(-12.5deg);} 22.2% {transform: skewX(6.25deg) skewY(6.25deg);} 33.3% {transform: skewX(-3.125deg) skewY(-3.125deg);} 44.4% {transform: skewX(1.5625deg) skewY(1.5625deg);} 55.5% {transform: skewX(-0.78125deg) skewY(-0.78125deg);} 66.6% {transform: skewX(0.390625deg) skewY(0.390625deg);} 77.7% {transform: none;} 100% {transform: none;} }

Blur In

Das Element erscheint und verschwindet mit einem Unschärfe-Effekt.

@keyframes blurIn { 0% {filter: blur(8px); opacity: 0;} 50% {filter: blur(0); opacity: 1;} 100% {filter: blur(8px); opacity: 0;} }

Slide Up

Das Element gleitet von unten nach oben ins Blickfeld.

@keyframes slideUp { from {transform: translateY(100px); opacity: 0;} to {transform: translateY(0); opacity: 1;} }

Pop

Das Element "poppt" als Skalierungseffekt hervor.

@keyframes pop { 0% {transform: scale(1);} 50% {transform: scale(1.3);} 100% {transform: scale(1);} }

Color Change

Das Element wechselt zyklisch die Farbe.

@keyframes colorChange { 0% {background-color: var(--primary);} 33% {background-color: var(--accent);} 66% {background-color: var(--secondary);} 100% {background-color: var(--primary);} }
